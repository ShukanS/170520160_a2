What is Browser Cache?



Browser Cache is a component in a browser's software that stores data and information from different websites visited by the user, via "cookies". For MailChimp, if the user has not yet visited the site, MailChimp will recognize that, as it will find that there is no cookie from that site in the browser cache, and thus display the pop-up. Once the user visits the website and the pop-up is shown, a cookie from the site will be planted into the browser cache. If, however, the site has been visited before, MailChimp will find the cookie in the browser cache, and as a result, the pop-up will not be shown. This is why it is important to clear the cache in order to test the MailChimp pop-up, as it wont show up if you dont, due to the presence of the cookie from the website in the browser cache.
